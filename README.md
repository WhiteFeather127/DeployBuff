# Buff效果-部署周期

## DeployBuff <主动>

位于 `rulesmd.ini`：

```ini
[SomeBuffType]
Effect.Type=Deploy
```

周期性触发单位的 **Deploy 能力**，可用于触发 `DeployFire=yes` 单位的部署开火，或执行步兵的部署/收起动画。

进入生效状态后，按照 **`Effect.Timer`** 指定的间隔（帧），反复触发单位的 Deploy 动作，总共触发 **`Effect.Counts`** 次后自动进入结束状态。

### 注意

- 如果单位没有部署能力（`DeployFire=yes` 或步兵部署逻辑），此效果不会产生任何作用。
- 部署间隔建议不小于 30 帧（约 1 秒），过短的间隔可能带来意外的行为或性能开销。

### 效果强度值转换：0 项参数

效果强度值对此效果种类无效。

### 效果种类相关属性

位于 `rulesmd.ini`：

```ini
[SomeBuffType]
; 效果种类相关属性
Effect.Timer=0                                 ; 整数 , 部署间隔 , 每 N 帧触发一次部署 , 0 = 每帧都触发（不推荐） , 默认值是 0 , 单位 : 帧
Effect.Counts=-1                               ; 整数 , 部署次数 , 次数耗尽会立刻进入结束状态 , 等于 0 会无法生效并直接进入结束状态（算作次数耗尽） , 负数 = 无限次 , 默认值是 -1 , 单位 : 次
Effect.Anims=                                  ; 动画 , 【触发部署时播放的动画】 , 不写就不显示动画
```

### 广播与监听相关属性

不支持广播或监听。

### 数值显示相关属性

不支持数值显示。
